09 / Discovery

Phantom's discovery inside the Revolut app.

Phantom is hosted within Revolut's Virtual Card section, right where the most likely adopters already are.

01 / Placement
Lives in Virtual Cards
Phantom is a new card type within Revolut's existing Virtual Card section. The nudge to use Phantom pops up right when a virtual card user is copying their card details to paste it on a merchant site to complete payment.
02 / Audience
Users who avidly use the virtual card feature
These are Revolut's most digitally-fluent users. They already chose virtual over physical. Switching from a standard virtual card to Phantom is the shortest adoption path possible, with zero learning curve.
03 / Adoption
Same ritual. Zero risk.
No new habit needed. The user copies a Phantom number instead of their real one. The action is identical, but now the number self-destructs the moment the transaction clears.
